Add RAGina to any website with a single <script> tag. She'll appear as a floating chat bubble, ready to answer questions from your documents.

For Developers

💻 Programmatic API

// Full control over RAGina RAGina.init({ indexUrl: './my-index.json' }); // Load custom data RAGina.loadData({ "page1.html": { bodyText: "Your content..." } }); // Load from folder input document.getElementById('folderPicker') .addEventListener('change', (e) => { RAGina.loadFolder(e.target.files); }); // Direct retrieval access const engine = RAGina.getEngine(); const chunks = engine.retrieve("your question");

What is RAGina?

RAGina is a tiny, vanilla JavaScript library that implements Retrieval‑Augmented Generation entirely in the browser. Point it at a directory of HTML files (or a pre‑built JSON index), and it instantly becomes an AI‑powered Q&A system grounded in your content — with no server, no API key, and no cost.

It combines a lightweight TF‑IDF search engine with a free LLM to answer questions strictly from your own documents. All processing happens client‑side; your data never leaves the browser except the anonymised prompt to the LLM.

How RAGina Works

The architecture is simple, yet powerful. Everything runs inside your browser:

1. Scraping & Chunking

When you select a folder, the browser reads each .html file, extracts the visible text inside <body>, and splits it into chunks of approximately 200 characters (broken at sentence boundaries).

2. Indexing with TF‑IDF

We build a classic Term Frequency – Inverse Document Frequency index. For each chunk, we count how often each word appears (TF), and for the whole collection, we calculate how rare each word is (IDF). This gives us a vector representation that can be compared with the user's query.

3. Retrieval

When you ask a question, the query is turned into a TF‑IDF vector and we compute the dot product with every chunk's vector. The top k chunks (default 3) with the highest score are selected as the most relevant context.

4. Augmented Generation

The retrieved chunks are formatted into a prompt together with the question and sent to a free text generation API. The LLM is instructed to answer only based on the provided context. The response is displayed along with the sources.

TF‑IDF Retrieval

function retrieve(query, topK = 3) {
  const qWords = query.toLowerCase().match(/\b\w+\b/g) || [];
  const qTF = {};
  qWords.forEach(w => qTF[w] = (qTF[w] || 0) + 1);

  const scores = chunks.map((ch, idx) => {
    const cWords = ch.text.toLowerCase().match(/\b\w+\b/g) || [];
    const cTF = {};
    cWords.forEach(w => cTF[w] = (cTF[w] || 0) + 1);
    let score = 0;
    for (const w of Object.keys(qTF)) {
      if (cTF[w] && idf[w]) score += qTF[w] * cTF[w] * idf[w];
    }
    return { idx, score };
  });
  scores.sort((a, b) => b.score - a.score);
  return scores.slice(0, topK).map(s => chunks[s.idx]);
}
